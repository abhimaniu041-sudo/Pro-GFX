# 🎮 Pro GFX - BGMI Optimizer

**Pro GFX** ek powerful Android utility hai jo BGMI (Battlegrounds Mobile India) ke liye graphics aur sensitivity ko optimize karti hai. Ise specifically Android 8.0+ devices ke liye design kiya gaya hai.

## ✨ Features
* **🎯 Smart Sensitivity:** Device tier (High/Mid/Low) ke hisaab se best sensitivity values.
* **🎨 Graphics Pro:** Smooth, Balanced, aur HDR mode unlocker.
* **⚙️ Advanced Mode:** Shizuku integration for system-level optimizations.
* **🟣 Floating Panel:** Game ke dauran quick settings access karne ke liye draggable overlay.
* **🚀 Core Utility:** RAM cleaner aur OBB core configuration manager.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI:** XML with ViewBinding
* **Architecture:** Modular Utility-based
* **API Support:** Shizuku API, System Overlay API

## 📥 Build & Installation
1.  GitHub **Actions** tab par jayein.
2.  Latest successful build se **APK** download karein.
3.  "Display over other apps" permission allow karein.
4.  Advanced features ke liye Shizuku ko activate karein.
